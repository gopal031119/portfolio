Crowd Computing Using CNN

A web app that uses a VGG-16 Convolutional Neural Network (CNN) to count people in images. Combining deep learning with image preprocessing techniques, the app provides an accurate count of people within the given images, making it useful for crowd analysis and monitoring.

Live Preview
Category
Machine Learning
Client
Academic Project

Project Description

This web app combines deep learning models with image processing to detect and count the number of people in a given image. By using a VGG-16 CNN, the application can accurately detect people in a variety of environments and conditions, making it useful for crowd monitoring in events or public spaces.

Technologies used: CNN, Django, TensorFlow, OpenCV.

The Story

The project was inspired by the need for real-time crowd monitoring and management. The system combines computer vision and deep learning to offer an automated and accurate solution for counting people in images, with a focus on simplicity and usability for users.

Our Approach

By training a VGG-16 CNN on diverse crowd images, the model was able to recognize and count people effectively. The backend, built in Django, communicates with the model, providing the front end with real-time data. OpenCV was used for preprocessing images and preparing them for analysis.

Smart Dustbin - Dry and Wet Waste Separator

An IoT-based smart dustbin that automatically separates dry and wet waste. The system uses sensors to detect the type of waste and directs it to the appropriate compartment, promoting better waste management and cleaner environments.

Live Preview
Category
IoT
Client
Academic Project

Project Description

The smart dustbin project is designed to solve waste management issues in urban areas. It uses IoT sensors to automatically detect the type of waste and separates them into dry and wet categories. It is an innovative solution to promote cleanliness and waste segregation in public spaces.

Technologies used: IoT, AWS, Sensors, Microcontrollers.

The Story

The project was born out of the need for better waste management systems in cities. With urban populations growing rapidly, waste segregation has become a critical challenge. The smart dustbin uses IoT technology to solve this problem by automating the process of waste separation.

Our Approach

We approached this project by designing a system that can differentiate between wet and dry waste using sensors. The backend was powered by AWS for cloud storage and data management, while the frontend allows users to monitor waste segregation in real-time.
